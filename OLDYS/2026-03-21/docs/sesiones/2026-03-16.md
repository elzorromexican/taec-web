## 🧠 Decisiones de diseño vigentes
> Estas reglas NO están en el código, hay que respetarlas siempre

- **Logos en fondos oscuros** → pill blanca (`background:#fff; border-radius:10px; padding:10px 20px; box-shadow:0 2px 12px rgba(0,0,0,.15)`) — NUNCA `filter: brightness(0) invert(1)`
- **Vyond colores** → naranja `#C84D16`, gradiente hero `#1a0a00 → #3d1200 → #C84D16`
- **Totara colores** → teal `#A8DBD9`, gradiente hero `#001f3d → #004775 → #0077b6`
- **Moodle colores** → naranja `#F98012` (base confirmada por cliente)
- **Articulate colores** → azul claro `#7DD3FC` para eyebrow, gradiente hero oscuro
- **Precios Vyond** → solo precio anual, sin precio mensual
- **Precios Articulate** → USD sin IVA, pago en MXN al tipo de cambio SAT/Banxico, factura CFDI
- **Nav tabs** → primer tab siempre es la marca padre, luego sub-productos
- **Videos hero** → autoplay muted + botón 🔇/🔊 esquina inferior derecha
- **Eyebrow** → texto grande en color de marca, encima del H1, mismo patrón en todas las páginas de producto
- **Badge reseller** → en cuadro derecho del hero, sobre los bullets — nunca en la sección de reconocimientos inferior
- **Ayleen AI (Totara)** → es feature integrado en toda la suite, NO producto separado
- **Neocities** → links de recursos apuntan a `taec-elearning.neocities.org` — dejar así por ahora
- **Lead magnet** → formulario gated (EmailJS) antes de revelar descarga — captura de leads

## 🐛 Bugs resueltos (no repetir)
- **Logos all-white** → pill blanca, NUNCA filter CSS
- **Tab activo negro sobre blanco** → `background:transparent!important` + `color` de marca `!important`
- **Rise 360 imagen bloqueada** → mockup CSS del editor (sidebar + canvas)
- **Sticky trigger inconsistente** → `window.scrollY > 600` en lugar de IntersectionObserver
- **GIF con caja blanca** → `mix-blend-mode: multiply` en contenedor circular de color de marca
- **G2 Totara 404** → URL era `/products/totara-learn/` → correcto: `/products/totara-lms/`
- **Dropdown hover gap** → `top: 100%` en CSS (antes `calc(100%+6px)`)
